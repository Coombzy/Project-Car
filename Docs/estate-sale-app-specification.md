# Estate Sale App - High-Level Specification

**Last Updated:** July 2026

## Overview
Separate business/app for estate liquidation services.

## Core Features
- Inventory organization and consistent photography guidance
- AI item identification and pricing suggestions
- Employee approval/confirmation workflow
- eBay listing automation
- Customer portal (view items, approve sales)
- Chat between employees, owners, and AI

## Integration
- Mission Control as the hub
- Shared agent fleet

## Tech
- Mobile-first
- Reuse components from Project Car where possible
