# Project Car - Integrated eBay Marketplace Specification

**Last Updated:** July 2026

## Overview
Integrated marketplace in Project Car for subscribers to list used/new parts.

## Features
- Subscriber listing of parts from their cars
- 10% commission to Project Car
- eBay integration
- Chat with employees/AI
- Customer portal for viewing listings

## Integration
- Part of Project Car app
- Mission Control for metrics and orchestration
