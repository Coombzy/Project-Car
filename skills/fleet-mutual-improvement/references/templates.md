# Templates — fleet mutual improvement

## AUDIT-PACK.md (minimum sections)

```markdown
# <Agent> git-safe mutual-audit pack

**Charter:** <one sentence — immutable job>
**Will not become:** <anti-role>
**Exported:** YYYY-MM-DD
**Hardware:** chip, RAM GB (no serials)

## Unique strengths (offers to peers if role-fit)
## Gaps (asks from peers)
## Local LLM / routing (if any)
## Always-on notes
## Discord notes
## Skills snapshot (counts + notable paths)
## Security affirmation (public-safe)
```
